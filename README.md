Natureza — Full demo ready for GitHub Pages

Files:
- index.html (single-file demo with Services table, contact, hero)
- CNAME (www.natureza.in)

Quick deploy:
1. Create a new GitHub repo and upload these files at the root.
2. In repo settings -> Pages -> set branch to main and folder to / (root).
3. Add CNAME to repo or in domain provider set CNAME for www to <your-username>.github.io
